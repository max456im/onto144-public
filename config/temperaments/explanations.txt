### Пояснение полей:

1. **`attention_bias`** (`f32`, 0.0–1.0)  
   Используется в `attention_model.rs` для определения:
   - начальной глубины причинного анализа,
   - скорости переключения между узлами в причинной сети,
   - веса социального контекста.  
   Высокое значение → глубокая, узкая обработка.  
   Низкое значение → широкая, поверхностная обработка.

2. **`inversion_preference`** (`string`)  
   Должен **точно соответствовать** целевому темпераменту из матрицы инверсии в `src/temperament/inversion_matrix.rs`:  
   - Choleric ↔ Phlegmatic  
   - Sanguine ↔ Melancholic  
   Это значение **не используется для переопределения логики**, а служит для:
   - логирования,
   - согласованности с конфигурацией,
   - будущей поддержки пользовательских модификаций (в рамках 144-профиля).

---

### Интеграция:

При загрузке профиля (`Profile::from_file`):
1. Система читает `element` и `temperament` из SGCL.
2. Загружает `config/elements/{element}.yaml` и `config/temperaments/{temperament}.yaml`.
3. Объединяет параметры в `SubjectiveInvariants`.
4. Использует их для:
   - расчёта энергетической утечки,
   - выбора режима внимания,
   - определения поведения при инверсии.
